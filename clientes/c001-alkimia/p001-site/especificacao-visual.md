## Tradução pra WEB (godaylight + Real, vestido de Alkimia)
- **Formato:** a referência é PDF 16:9 landscape; a gente faz WEB scroll (godaylight). Cada PRODUTO vira uma SEÇÃO rica de scroll; os SPREADS editoriais viram seções full-bleed entre produtos.
- **Site (index.html):** estilo godaylight — full-bleed, scroll-storytelling, copy curta, CTAs multi-toque — claro/off-white, product/process-forward.
- **Catálogo (catalogo.html):** grid de produtos → página de produto rica (padrão da referência acima, adaptado a P&B e scroll) → ficha/condições + créditos no fim. Página oculta (noindex).
